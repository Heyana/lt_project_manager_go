## 技术栈对比

### Electron 版本

- 主进程: Node.js + Express
- 渲染进程: Vue3 + TSX
- 数据存储: lowdb (JSON)
- 资源访问: 本地文件系统

### Go Web 版本

- 后端: Go + Gin
- 前端: Vue3 + TSX + Vue Router
- 数据存储: SQLite + GORM
- 资源访问: CDN 服务

## 核心差异处理

### 1. 窗口管理 → 路由管理 ✅

```typescript
// Electron: 创建新窗口
ipcRenderer.invoke("create-window", { url, name });

// Web: 路由跳转
router.push(`/preview/${id}`);
```

### 2. 本地文件访问 → CDN 访问 ✅

```typescript
// Electron: 本地路径
const filePath = path.join(assetsPath, link);

// Web: CDN URL
const fileUrl = `${config.cdnBaseUrl}/${link}`;
```

### 3. IPC 通信 → HTTP API ✅

```typescript
// Electron: IPC
const result = await ipcRenderer.invoke("get-projects");

// Web: HTTP
const result = await axios.get("/api/projects");
```
