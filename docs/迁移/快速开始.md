# 快速开始 - 零拓项目管理器 Web 版

## 项目概述

这是零拓项目管理器从 Electron 桌面应用迁移到 Go Web 应用的版本。

### 技术栈

- 后端: Go + Gin + SQLite + GORM
- 前端: Vue3 + TypeScript + TSX + Vue Router + Axios
- 构建: Vite

## 开发环境启动

### 1. 启动后端服务

```bash
cd lt_project_manager_go
go run -tags dev .
```

后端服务将启动在: http://localhost:23370

- API 文档: http://localhost:23370/api/docs
- CDN 服务: http://localhost:23370/cdn/

### 2. 启动前端开发服务器

```bash
cd lt_project_manager_go/frontend
yarn install  # 首次运行需要安装依赖
yarn dev
```

前端服务将启动在: http://localhost:5174

### 3. 访问应用

打开浏览器访问: http://localhost:5174

## API 接口

### 项目管理

```bash
# 获取项目列表
GET /api/projects?type=all&page=1&page_size=20

# 创建项目
POST /api/projects
{
  "name": "项目名称",
  "type": "table",
  "link_type": "url",
  "link": "https://example.com",
  "description": "项目描述",
  "sort_order": 1
}

# 获取项目详情
GET /api/projects/:id

# 更新项目
PUT /api/projects/:id

# 删除项目
DELETE /api/projects/:id

# 搜索项目
GET /api/projects/search?keyword=关键词&type=table

# 更新排序
PUT /api/projects/:id/sort
{
  "sort_order": 10
}
```

## 开发指南

### 添加新的 API

1. 在 `models/` 中定义数据模型
2. 在 `services/` 中实现业务逻辑
3. 在 `controllers/` 中创建控制器
4. 在 `api/routes.go` 中注册路由

### 添加新的前端页面

1. 在 `frontend/src/views/` 中创建页面组件
2. 在 `frontend/src/router/index.ts` 中添加路由
3. 在 `frontend/src/api/` 中封装 API 调用

## 常见问题

### Q: 后端启动失败?

A: 确保端口 23370 没有被占用,检查 `config.yaml` 配置
